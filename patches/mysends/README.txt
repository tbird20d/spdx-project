This 'mysends' directory contains scripts which were
used to send SPDX patches to the kernel mailing lists.

They are useful to save because each patch (or set of
patches) may have unique recipient lists, which are
encoded in the scripts.  It sometimes takes weeks
before I hear back on a whether a patch was accepted.
Keeping a script around allows to quickly re-spin
a patch based on feedback, and send to the same
people.
