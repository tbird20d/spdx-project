# Tim Bird's SPDX Project #

This project has has series of tools, notes and data for working
on adding SPDX-License-Identifier lines to the Linux kernel source files.

These tools were useful for me (Tim Bird) in analyzing files that
are missing an SPDX ID line, discovering what license to apply,
applying missing license lines, upstreaming the patches created,
and testing the kernel for files that are highest priority to
be worked on next.

See NOTES files in the 'doc' directory for descriptions of
how to use the tools, problem areas. etc.

Also, you can find test results, guidelines for working on
this task, historical kernel license information, and more
at https://birdcloud.org/bc/SPDX_Project
